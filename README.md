# 172-masata-masyarakatsadarwisata
Landing page prototype untuk organisasi Masyarakat Sadar Wisata (MASATA).

## Fitur Utama
- Navigasi responsif (desktop dan mobile) dengan tampilan modern berbasis Tailwind CSS.
- Section konten lengkap: Beranda, Tentang Kami, Blog, Struktur Organisasi, dan CTA gabung.
- Galeri kegiatan interaktif dengan modal detail dan tombol berbagi ke media sosial.
- Peta sebaran DPC Kab/Kota interaktif menggunakan Leaflet JS dan OpenStreetMap.
- Popup disclaimer awal serta tautan ke kanal resmi MASATA.
